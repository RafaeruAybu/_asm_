### AT&T asm.

##### to build use:
> make

##### to clean use:
> make clean

##### to build and execute(in folders):
> make && ./\<name of exec\>

### Some notes about asm(in asm style :KEKW:):

##### consts:
>    $name

##### registers:
>   %edx
    eax
    ebx
    ecx
    edx
    esi
    edi

##### commads:
>   mov
    cal
    int
    jmp

##### data types:
>   ascii "......."
    word
    long
    byte

##### example:
>   mov $5,%eax
